Add section to menu bar

Interactive Navigation Menu Project

This project demonstrates a modern, interactive navigation menu for a website, designed to enhance user experience and provide dynamic customization. The navigation bar is fixed at the top of the page, ensuring that users always have access to the main sections of the site, regardless of their scroll position. Users can add new sections to the navigation menu by entering a section name and selecting a custom hover color from a dropdown menu. When the "Add" button is clicked, the new section is saved and immediately appears in the navigation bar after the page reloads. The selected color is used as the hover effect for that specific menu item, making it visually distinct and easy to identify.

The navigation bar also features a scroll effect: as the user scrolls down the page, the background color and style of the navbar change, providing a clear visual cue that the user is no longer at the top of the page. This effect is achieved using JavaScript, which listens for scroll events and updates the navbar's class accordingly. The menu is fully responsive and remains visible on all pages, making navigation seamless. The project leverages HTML for structure, CSS for styling and layout, and JavaScript for interactivity and state management. All custom menu items are stored in the browser's localStorage, ensuring that user preferences persist across sessions. This project is ideal for learning about DOM manipulation, event handling, and creating user-friendly, interactive web interfaces.
